# Keylogger

**Keylogger** es una herramienta que registra las teclas presionadas en el sistema. Este proyecto está destinado para fines educativos y de investigación en seguridad informática.

## Características

- Registra las teclas presionadas y las guarda en un archivo.
- Compatible con sistemas operativos **Windows** y **Linux**.
- Realizado en Python 3.
